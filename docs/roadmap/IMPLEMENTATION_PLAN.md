## 长期扩展路线

1. **消费者权益代理**：订阅识别、自动取消、账单异常检测、退款统计
2. **购物避坑助手**：商品分析、评价真假判断、退货窗口提醒
3. **家庭文件 OS**：共享文件库、证件到期提醒、房租合同管理
4. **老人防骗模式**：大字体 UI、一键发给家人确认、高风险弹窗
5. **自动执行代理**：邮件发送、表单填写、浏览器自动化、自动跟进
